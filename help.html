使用說明

篩選表達式、排序表達式

如1+1、2*3等，是用來篩選和排序貓咪的運算式，類似數學的表達式，在程式語言裡通常稱做expression。

  • 用在篩選上 篩選表達式將會算出一個結果，如果結果成立(true)，則會將該結果移除；如果是不成立(false)，則結果將會保留

    例如 range >= 400 篩選距離大於 400 的貓咪 例如 (dps > 40000) && (price <= 5000) 篩選DPS大於40000且價格小於5000的貓咪

  • 用在排序上 排序表達式將會算出一個數值，會以這個數值將篩選過的結果做由大到小的排序

    如果你沒有指定排序表達式，將會以貓咪的ID大小作為預設排序

    例如，輸入 dps 將會將dps由大到小排序

    例如，dps/cd 就是每秒的DPS產量(ADPS)

    例如，dps*range 就是DPS和距離的乘積

    如果看數值最小的排序，可以加上負號，如 -dps 就是從DPS最小排到最大

常數

有些預先定義的符號，稱為常數

如 RED 代表紅色敵人

如 AB_WAVE 代表波動能力

這些符號不用特別去記憶，點擊篩選按鈕就可以看到

運算子

類似數學中的運算式，可以用來作數值的運算和邏輯運算

運算子解釋最先結合順序(最高1)
*乘法1
/除法1
%取餘數1
+加法2
-減法2
>大於3
>=大於等於3
<小於3
<=小於等於3
==等於4
!=不等於4
&位元AND4
|位元OR4
^位元XOR4
&&邏輯AND5
||邏輯OR5

位元AND、位元OR、位元XOR平常較少用到，如果要用AND功能，請用&&，如果要用OR功能，請用||

另外 ( 和 ) 可以指定優先運算，如 -(1+2)*3 是 -9

  • pow: pow(x, y)是x的次方
